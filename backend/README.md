# Job Portal + Applicant Tracking System

A full-stack job portal with ATS capabilities built with Rails and React.

## Features

- ✅ Job postings management
- ✅ Resume upload and parsing
- ✅ Employer dashboard with analytics
- ✅ Applicant tracking system
- ✅ Email notifications
- ✅ Advanced search & filtering
- ✅ Role-based access control

## Quick Start

### Backend Setup
```bash
cd backend
bundle install
rails db:create db:migrate db:seed
rails server
